verigaNET BLOCK is very lean and lightweight blockchain as a service managed by verigaNET.

verigaNET BLOCK is based on XNO blockchain technology.

Source code of XNO can be found at Github

By launching blockchain on verigaNET BLOCK you will get fully independent blockchain managed by us.

Beside the main blockchain, verigaNET BLOCK will also deploy following components:

  • Web Wallet App: Ready to use wallet app served over web
  • Block Explorer: Ready to use block explorer
  • Peering Nodes: Initial peering nodes to bootstrap and share bootstrap information with new nodes in blockchain network.
  • Representative Nodes: Initial principle representative nodes.
  • REST API Service: Native REST API endpoint service to interact with blockchain.
  • Binaries: Binaries to run nodes at outside verigaNET infrastructure.
